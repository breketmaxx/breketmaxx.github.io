# VS Мебель - Веб-сайт

Веб-сайт для мебельной компании с мобильной адаптацией.

## Технологии

- React 18
- React Router DOM 6
- Vite
- CSS3 (с мобильной адаптацией)

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run dev
```

Проект будет доступен по адресу http://localhost:5173

## Сборка для продакшена

```bash
npm run build
```

## Структура проекта

```
src/
├── components/      # Компоненты (Layout, Navigation)
├── pages/          # Страницы приложения
│   ├── Home.jsx
│   ├── Services.jsx
│   ├── Calculator.jsx
│   ├── Portfolio.jsx
│   ├── About.jsx
│   └── Contacts.jsx
├── App.jsx         # Главный компонент с роутингом
└── main.jsx        # Точка входа
```

## Страницы

- **Главная** (`/`) - Главная страница с приветствием и преимуществами
- **Услуги** (`/services`) - Список предоставляемых услуг
- **Калькулятор** (`/calculator`) - Калькулятор стоимости мебели
- **Портфолио** (`/portfolio`) - Галерея выполненных работ
- **О компании** (`/about`) - Информация о компании
- **Контакты** (`/contacts`) - Контактная информация и форма обратной связи
